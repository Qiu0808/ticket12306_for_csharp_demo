12306订票客户端 FOR .NET 演示项目
========

本项目是由[木鱼]发起的一个基于.NET4.0的[12306]订票客户端。项目使用C#语言，使用[FSLib.Network]项目作为网络层访问库，主要用于演示目的。

本项目仅包含基本的12306功能实现：

* 登录
* 查询
* 提交订单

更详细进阶的功能将会在之后视情况是否加入。

本项目仅作为演示目的，禁止用于将全部或部分代码用于商业目的。
禁止转载。
